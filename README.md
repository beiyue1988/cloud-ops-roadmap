# cloud-ops-roadmap

## 项目定位

`cloud-ops-roadmap` 是面向零基础学习者的云计算运维成长知识库，目标是同时服务于课程学习、企业培训和长期知识维护。完整范围、阶段划分与质量标准以 [CODEX-PRD.md](CODEX-PRD.md) 为准。

## 当前状态

Phase 1“导航与完整大纲”已通过[最终验收](项目管理/验收记录/Phase-1-最终验收报告.md)，形成 21 阶段、620 章和 817 条直接前置的稳定基线。当前只解锁 Phase 2 的任务设计、样章门、模板适配与阶段 00–05 边界化单任务；正式学习正文尚未获批批量生产。

下一阶段主会话从 [Phase 2 主会话交接](项目管理/会话交接/Phase-2-主会话交接.md)启动。

项目官方仓库：[beiyue1988/cloud-ops-roadmap](https://github.com/beiyue1988/cloud-ops-roadmap)。任务进度以[项目管理/任务看板.md](项目管理/任务看板.md)为准。

## 权威规范

## 参与方式

所有工作必须先取得边界明确的任务包。执行会话只修改任务包授权的文件，实际运行规定验证并提交执行报告；主会话负责规格、状态更新和最终验收。禁止把密码、Token、Cookie、私钥或云 AccessKey 提交到仓库。
