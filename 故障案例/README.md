# 故障案例

## 目录职责

本目录组织基于真实现象、证据、根因与恢复验证的独立故障案例。

## 权威内容类型

未来故障案例的现象、影响、证据链、根因、恢复、验证和预防材料以本目录为权威归属。

## 非范围

- 不复制知识原理正文；
- 不替代实验手册中的故障注入或项目中的场景演练；
- 本任务不创建故障案例、处置命令或答案。

## 命名规则

- 目录入口固定为 `README.md`；
- 后续案例文件名不含空格并以短横线连接，技术名保留英文，具体编号由后续任务确定。

## 上游与下游

- 知识来源：[知识库](../知识库/README.md)
- 场景验证：[实验手册](../实验手册/README.md)
- 业务背景：[项目实战](../项目实战/README.md)
- 能力表达：[面试体系](../面试体系/README.md)

## 当前阶段
