# P0-04 仓库校验工具实施计划

> **面向 AI 代理的工作者：** P0-04 执行会话必须使用 `executing-plans` 与 `test-driven-development` 技能逐任务实现本计划，并用复选框跟踪进度；提交前必须使用 `verification-before-completion`。本项目要求具体实现与独立审查交给不同新会话，主会话只签发任务、初审、解锁和验收。

**目标：** 创建一个仅依赖 Python 3.11 标准库、具有稳定错误编号和完整单元测试的本地仓库 Markdown 校验器，并通过独立审查成为后续内容任务的机械门禁。

**架构：** `validate_repository.py` 以规则常量、文本预处理、单文件规则、仓库聚合和 CLI 五层纯函数组织；`test_validate_repository.py` 使用临时仓库覆盖 33 个规定行为；README 只说明实际功能和限制。P0-04 执行、主会话范围初审、P0-04B 独立审查与主会话最终验收严格串行。

**技术栈：** Python 3.11.15 标准库（`argparse`、`dataclasses`、`datetime`、`pathlib`、`re`、`sys`、`tempfile`、`unittest`、`urllib.parse`）、PowerShell、Git。

**批准设计：** [P0-04 仓库校验工具设计说明](./2026-07-11-P0-04-仓库校验工具-设计说明.md)，设计提交 `442bdb626a0f714d290e65f6d6af6a365e5fd7a9`。

## 3. 诊断格式与辅助结构

内部诊断建议使用不可变 dataclass，最后统一格式化：

```python
@dataclass(frozen=True, order=True)
class Diagnostic:
    path: str
    line: int
    rule: str
    message: str

    def render(self) -> str:
        return f"{self.path}:{self.line}: {self.rule} {self.message}"
```

公开单文件函数仍返回 `list[str]`；内部辅助函数返回 `list[Diagnostic]`。如果执行会话选择让公开函数直接构造字符串，必须用统一 `_error_sort_key()` 解析路径、数字行号、规则编号和消息，不能依赖纯字符串排序导致第 10 行排在第 2 行之前。

仓库根错误使用 `.:0`。路径必须相对根目录并统一为 `/`。敏感信息诊断不得回显匹配值。

```python
FENCE_RE = re.compile(r"^(?P<indent> {0,3})(?P<fence>`{3,}|~{3,}).*$")

def _mask_fenced_code(text: str) -> str:
    lines = text.splitlines(keepends=True)
    active_char: str | None = None
    active_length = 0
    masked: list[str] = []
    for line in lines:
        match = FENCE_RE.match(line.rstrip("\r\n"))
        if active_char is None and match:
            fence = match.group("fence")
            active_char, active_length = fence[0], len(fence)
            masked.append("\n" if line.endswith("\n") else "")
            continue
        if active_char is not None:
            stripped = line.lstrip()
            if stripped.startswith(active_char * active_length):
                active_char = None
                active_length = 0
            masked.append("\n" if line.endswith("\n") else "")
            continue
        masked.append(line)
    return "".join(masked)
```
