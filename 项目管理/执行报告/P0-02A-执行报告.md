## 9. 异常、风险、权限请求与未决问题

### 异常

1. 首次相对链接内联检查因 JavaScript 向 PowerShell 传递正则时反斜杠被消耗，错误显示相对链接为 0。该问题只影响临时只读命令，不影响仓库文件；定位根因后改用原始字符串，最小复现从 0 恢复为 README 命中 11，最终全量复跑为相对链接 15、失效 0。
2. 首次 `git add` 在沙箱内因无法创建 `.git/index.lock` 退出 1；随后按授权流程请求 Git 元数据写权限，获批后以相同显式文件列表执行成功。
3. 执行报告预提交校验的首次调用被 Markdown 反引号截断 JavaScript 模板，命令未执行且仓库无变化；移除易冲突字面量后重新运行。
4. 已提交规则的首次复核误用 `Measure-Object -Line`，因其不计 29 个空行而得到 197；对比原生输出数组、工作区文件和 Git blob 后确认三者内容一致，改用任务规定的数组行数口径得到 226。
