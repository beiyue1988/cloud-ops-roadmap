## 4. 最小修复

生产实现只修改一行：

```python
HTML_BREAK_RE = re.compile(r"<br(?=[\s/>])", re.IGNORECASE)
```

修复语义：

1. 大小写不敏感地识别 `<br`；
2. 紧随字符只有为空白、`/` 或 `>` 时才确认标签名；
3. 确认标签名后不扫描、不解析属性体，因此属性值中的 `<`、`>`、单双引号或其他内容不影响识别；
4. `a`、`-`、`:`、`x` 等不是允许分隔符，因此 `<bracket>`、`<br-description>`、`<br:custom>`、`<brx>` 不匹配；
5. 章节标题和主要目标仍由 `OL005` 负责，派生文本字段仍由 `OL002` 负责。

没有修改围栏/缩进代码屏蔽、迭代依赖图、项目总视图可比较状态、OL 编号或责任、CLI、公开函数、表头、字段、默认行为或 README。
