# 执行报告

本目录保存执行会话对单一任务包的交付证据。文件使用 `<任务编号>-执行报告.md` 命名，编号必须与任务包一致。

## 允许编辑者

执行会话只可创建任务包授权的报告；主会话可在验收时引用报告，但不以执行报告代替独立检查。其他会话不得覆盖已有报告。

## 必需字段

报告必须包含任务编号与结论、文件清单、逐项验收证据、实际命令、退出码和关键输出、官方来源与核验日期、提交哈希、权限请求或异常、风险、未解决问题和状态建议。

## 状态责任

执行报告只能建议 `submitted`、`blocked` 或返工需求；正式任务状态和是否 `accepted` 由主会话更新。报告不得保存凭据或敏感数据。
