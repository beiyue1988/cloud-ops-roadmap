## 目标与学习者价值

建立统一但不僵化的模板基线，使后续不同执行会话能够：

- 为完全零基础学习者生产一致、清晰、可验证的学习材料；
- 根据知识类型选择结构，避免所有章节机械包含安装、实验、故障和配图；
- 统一背景、实践、来源、证据、安全、成本和清理边界；
- 用稳定字段支持 P0-04 校验工具和主会话验收；
- 保持执行、独立审查和最终验收职责分离。

### 占位符、禁止事项和 HTML 注释

```powershell
$templateFiles = Get-ChildItem -LiteralPath '模板' -File -Filter '*.md' | Where-Object { $_.Name -ne 'README.md' }
foreach ($file in $templateFiles) {
  $text = Get-Content -LiteralPath $file.FullName -Raw -Encoding UTF8
  if ($text -notmatch '<必填：[^>]+>') { Write-Error "$($file.Name) 缺少必填占位符"; exit 1 }
  if ($text -notmatch '<!--[\s\S]*?-->') { Write-Error "$($file.Name) 缺少 HTML 填写说明"; exit 1 }
  if ($text -notmatch '禁止|不得') { Write-Error "$($file.Name) 缺少禁止事项"; exit 1 }
}
$badPlaceholder = rg -n 'TODO|TBD|待补充|稍后完善|这里填写|<填写|<替换' '模板'
if ($LASTEXITCODE -eq 0) { $badPlaceholder; exit 1 }
if ($LASTEXITCODE -ne 1) { exit $LASTEXITCODE }
```

## 人工检查

必须逐项记录“通过/不通过”和文件证据：

1. 十个模板可直接复制；
2. HTML 注释不会替代必填字段；
3. 知识模板不会诱导机械填充所有可选模块；
4. 实验模板对零基础学习者具有风险提示和可恢复性；
5. 图表模板服务理解并具备可访问性；
6. 故障模板体现证据驱动排查；
7. 面试模板不鼓励虚构生产经验；
8. 治理模板保持执行、审查和最终验收分离；
9. 模板没有成为第二份 PRD 或 AI 规则；
10. 模板没有正式内容、动态版本或伪造结果。

## 提交要求

必须形成两个独立提交：

1. 11 个模板目录文件：`docs: add project content templates`；
2. P0-03 执行报告：`docs: add P0-03 execution report`。

禁止 amend 已引用历史、禁止 rebase、禁止 push。第一提交不得包含报告，第二提交不得包含模板修改。

## 执行报告要求

`项目管理/执行报告/P0-03-执行报告.md` 必须包含：

1. 任务编号、日期、角色和建议状态；
2. 12 个白名单文件处理情况；
3. 模板提交完整哈希，且不预测报告提交自身哈希；
4. 10 类模板、11 个模板目录文件和字段检查矩阵；
5. 自动验证命令、退出码和关键输出；
6. 十项人工检查与证据；
7. 来源、版本入口、环境和证据等级说明；
8. 异常、权限请求、风险和未决问题；
9. 未 push、未创建 PR、未发布网站、未修改 GitHub 设置的确认；
10. 建议状态 `submitted`，不得宣布 `accepted`。

## 验收标准

1. 十类模板和 README 路径精确存在；
2. 图表模板作为独立第十类模板存在；
3. 占位符、字段、状态和证据术语一致；
4. 知识章节模板包含 13 键、九种类型、四种状态和八项共享骨架；
5. 知识模板自适应且没有固定安装模块；
6. 实验、项目、故障、面试和图表模板符合批准设计；
7. 四类治理模板符合 PRD 与 AGENTS；
8. 正式内容、动态版本、敏感信息和伪造证据命中为 0；
9. 相对链接失效为 0，写入白名单缺失/越界为 0；
10. 自动检查和人工检查均完成；
11. 两个提交范围准确，最终工作区干净；
12. 全过程未执行未经授权的远程写操作。

## 停止条件

- 规格、状态、分支、远程或工作区与任务包不符；
- `模板/` 或 P0-03 执行报告已存在；
- 需要写入白名单外文件；
- 需要改变 PRD、AI 规则、版本基线、目录基线或任务状态；
- 两类批准占位符无法满足模板需求；
- 验证无法在白名单内修复；
- 发现敏感信息或许可证风险；
- 无法形成规定的两个独立提交；
- 同一外部阻塞连续出现三次。
