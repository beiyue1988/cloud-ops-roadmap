# 项目管理

本目录保存任务状态、决策、执行计划、任务包、执行报告、验收记录和内容审计证据。`任务看板.md` 是任务状态的唯一权威来源。

## 编号与目录

- 任务使用 `P<Phase>-<序号>`，例如 `P0-01`。
- 决策使用 `ADR-<四位序号>`。
- 执行计划、任务包、执行报告和验收记录必须能够追溯到任务或 Phase。

## 允许编辑者

主会话维护计划、任务包、任务看板、正式决策和最终验收；执行会话只创建任务包明确授权的交付物与执行报告；独立审查会话只创建被授权的审查或验收记录。

## 必需信息与状态责任

每项管理记录至少包含编号、目标或结论、文件范围、证据、风险、责任角色和日期。执行会话只能建议状态，不能直接更新任务看板；任务状态、决策状态和验收结论由主会话更新。任何管理文件都不得保存密码、Token、Cookie、私钥、云 AccessKey 或其他敏感数据。
