# 阶段检查点

## 使用方式

阶段检查点把每个阶段的全量闭环章、该阶段章节清单已登记的实验深度和一个可观察能力结果连接起来。所需章节取对应阶段章节清单的最后一章；实践锚点只表示未来实践应达到的深度，不代表实验正文已经存在或完成。
