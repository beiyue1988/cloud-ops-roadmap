# 学习路线

## 目录职责

本目录按成长顺序组织知识入口，为不同学习目标提供阶段导航。

## 权威内容类型

- 阶段顺序、学习优先级与路线入口；
- 指向正式知识、实验和项目的导航链接。

本目录仅导航，不复制正文；正式知识正文以[知识库](../知识库/README.md)为唯一权威来源。

## 非范围

- 不保存正式知识正文；
- 不保存实验步骤、项目实现、故障案例或面试答案；
- 不在本任务中创建具体学习路线文件。

## 命名规则

- 目录入口固定为 `README.md`；
- 后续导航文件遵循仓库的无空格、短横线连接规则，并使用编号表达阅读顺序。

## 上游与下游

- 上游：[仓库说明](../README.md)
- 正文来源：[知识库](../知识库/README.md)
- 查询入口：[技术索引](../技术索引/README.md)
- 实践入口：[实验手册](../实验手册/README.md)
- 项目入口：[项目实战](../项目实战/README.md)
