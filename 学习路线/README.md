# 学习路线

## 目录职责

本目录把 21 个阶段的权威章节清单组织成面向不同目标的学习视图。路线用于选择顺序、优先级和节奏，不成为第二份知识正文。

章节 ID、直接前置和就业标签以[知识库](../知识库/README.md)及各阶段 README 为权威来源。路线中的短说明只帮助导航；需要查看完整章节表时回到对应阶段 README。

## 路线入口

- [学习路线导航](./00-学习路线导航.md)：先判断应选择完整成长、十二周就业优先级还是职业继续成长。
- [完整成长路线](./01-完整成长路线.md)：按 00–20 共 21 个阶段建立完整能力顺序。
- [三个月就业路线](./02-三个月就业路线.md)：按十二周能力进度呈现 293 章主线与 74 章弹性集合。
- [职业发展路线](./03-职业发展路线.md)：按七个能力层级查看阶段与岗位方向的关系。
- [每日每周学习模板](./04-每日每周学习模板.md)：按个人可用时间缩放日、周节奏。
- [知识前置依赖](./05-知识前置依赖.md)：查看全部 52 条跨阶段直接前置边。

## 使用边界

- 十二周表示能力进度，不承诺固定日历工时、掌握时长或就业结果。
- 路线只引用章节 ID 和短导航信息，不复制章节目标、原理、命令、配置或实验步骤。
- 实践锚点仅引用阶段清单已有的 `CP-*` 和 `LAB-L1` 至 `LAB-L4`；项目里程碑由后续波次定义。
- 章节清单发生经批准的变化时，必须从权威阶段 README 重新派生并原子更新受影响视图。
