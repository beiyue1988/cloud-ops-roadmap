# Level 1 基础实验

`LAB-L1` 定义单知识点的观察与验证：学习者在单一对象、低风险且可快速恢复的范围内完成操作，并说明观察结果与该知识点的关系。

## 准入

- 已学习目标知识点及实验声明的直接前置；
- 能识别目标对象、允许动作、成功信号和停止条件；
- 已准备隔离环境或等效的快速恢复方式。

## 能力边界

只验证一个主要知识点和一个主要对象。多组件部署、跨节点运行维护、受控故障处理以及架构方案比较不属于本级完成范围。

## 退出条件

学习者能够独立完成目标操作，取得可复验状态，并用前后变化说明结果；失败时能够停止并恢复到已知状态。

## 风险边界

默认使用非生产、无真实数据的隔离环境。目标范围不清、需要扩大权限、恢复条件不成立或可能持续计费时，不得继续。

## 证据

允许的证据类型为操作记录、状态输出和前后对比。证据必须标明环境与观察对象，不得把示意输出写成实际运行结果。
