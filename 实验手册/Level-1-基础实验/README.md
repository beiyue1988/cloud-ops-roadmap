# Level 1 基础实验

## 目录职责

本目录承接面向单个知识点的基础操作与验证类实验。

## 权威内容类型

未来只收录能够独立操作并验证单个知识点的 Level 1 实验。

## 非范围

- 不收录多组件综合部署、企业故障处置或架构设计实验；
- 不保存知识正文、脚本库或项目实现；
- 当前不含任何实验步骤。

## 命名规则

- 等级目录名固定为 `Level-1-基础实验`，入口固定为 `README.md`；
- 后续实验文件名不含空格并以短横线连接，具体编号由后续任务确定。
