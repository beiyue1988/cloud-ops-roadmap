# Level 2 综合实验

## 目录职责

本目录承接组合多个知识点完成部署目标的综合实验。

## 权威内容类型

未来只收录需要组合多个知识点并形成可验证部署结果的 Level 2 实验。

## 非范围

- 不收录单知识点练习、以故障维护为主的实验或架构方案设计；
- 不保存知识正文、脚本库或贯穿项目实现；
- 当前不含任何实验步骤。

- 后续实验文件名不含空格并以短横线连接，具体编号由后续任务确定。
