# 实验手册

## 目录职责

本目录按实践复杂度组织可复现实验入口，并关联相应知识阶段与项目能力。

## 权威内容类型

- Level 1：操作和验证单个知识点；
- Level 2：组合多个知识点完成部署；
- Level 3：维护并处理故障；
- Level 4：设计方案并解释取舍。

未来实验正文以对应 Level 目录为权威归属。

## 非范围

- 不保存正式知识正文；
- 不保存贯穿项目实现、独立故障案例或面试答案；
- 本任务不创建实验正文、命令、配置或结果记录。

## 命名规则

- 等级目录固定使用 `Level-N-中文名称`；
- 目录入口固定为 `README.md`；
- 后续实验文件遵循无空格、短横线连接规则，具体编号由后续任务确定。
