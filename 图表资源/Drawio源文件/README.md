# Drawio 源文件

## 目录职责

本目录保存未来正式架构图的可编辑 Draw.io 源文件。

## 权威内容类型

未来 `.drawio` 文件是对应正式架构图的唯一可编辑源。

## 非范围

- 不保存导出的 SVG、PNG、JPEG 或 WebP 文件；
- 不保存内嵌于 Markdown 的 Mermaid 图；
- 本任务不创建任何图表源文件。

## 命名规则

- 源文件使用 `.drawio` 扩展名；
- 与对应导出图保持同名，名称不含空格并以短横线连接；
- 目录入口固定为 `README.md`。

## 上游与下游

- 上游：[图表资源](../README.md)
- 导出归属：[导出图](../导出图/README.md)
- 内容来源：[知识库](../../知识库/README.md)
