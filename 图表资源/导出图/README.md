# 导出图

## 目录职责

本目录保存从正式可编辑源导出的、供 Markdown 引用的图像文件。

## 权威内容类型

未来导出的 SVG 及按需生成的其他图像格式以本目录为引用归属；可编辑权威源仍位于 Drawio 源文件目录。

## 非范围

- 不保存 `.drawio` 可编辑源文件；
- 不保存内嵌于 Markdown 的 Mermaid 图；
- 本任务不创建任何导出图。

## 命名规则

- 导出图与对应 `.drawio` 源文件保持同名；
- 名称不含空格并以短横线连接，扩展名按导出格式确定；
- 目录入口固定为 `README.md`。

## 上游与下游

- 上游：[图表资源](../README.md)
- 可编辑源：[Drawio 源文件](../Drawio源文件/README.md)
- 内容来源：[知识库](../../知识库/README.md)
