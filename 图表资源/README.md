# 图表资源

## 目录职责

本目录管理正式架构图的可编辑源文件与导出图之间的对应关系。

## 权威内容类型

- Draw.io 可编辑源文件以[Drawio 源文件](./Drawio源文件/README.md)目录为权威归属；
- 供 Markdown 引用的导出图以[导出图](./导出图/README.md)目录为发布归属。

简单 Mermaid 图未来仍内嵌在对应 Markdown 正文中，不在本目录单独保存。

## 非范围

- 不保存知识正文、实验步骤或项目代码；
- 不把导出图作为可编辑源文件；
- 本任务不创建 Draw.io、SVG 或其他图表文件。

## 命名规则

- 源文件与导出图保持同名，扩展名按格式区分；
- 名称不含空格并以短横线连接，技术名保留英文；
- 目录入口固定为 `README.md`。

## 上游与下游

- 内容来源：[知识库](../知识库/README.md)
- 项目来源：[项目实战](../项目实战/README.md)
- 源文件：[Drawio 源文件](./Drawio源文件/README.md)
- 导出文件：[导出图](./导出图/README.md)

## 当前阶段

Phase 0 仅建立图表所有权、格式边界与导航，不创建任何图表。
