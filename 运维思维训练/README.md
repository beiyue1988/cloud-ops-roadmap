# 运维思维训练

## 目录职责

本目录组织跨技术主题的排查、变更、复盘、容量、依赖与风险意识训练。

## 权威内容类型

未来分层排查、证据记录、变更控制、复盘改进和工程决策训练以本目录为权威归属。

## 非范围

- 不复制技术知识正文、企业制度或故障案例；
- 不替代实验和项目的实际验证；
- 本任务不创建训练题、案例或操作步骤。

## 命名规则

- 目录入口固定为 `README.md`；
- 后续文件名按训练主题组织，不含空格并以短横线连接，技术名保留英文。
