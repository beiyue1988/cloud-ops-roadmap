# 参考资料

## 目录职责

本目录组织跨章节复用的官方资料、标准、RFC 与来源索引。

## 权威内容类型

未来跨内容域的来源清单、引用归属与资料索引以本目录为权威归属；具体技术章节仍应保留直接来源链接。

## 非范围

- 不复制外部文档或保存无许可证依据的材料；
- 不替代知识正文中的具体来源说明；
- 本任务不创建资料清单、摘要或下载文件。

## 命名规则

- 目录入口固定为 `README.md`；
- 后续文件名按资料领域组织，不含空格并以短横线连接，标准和技术名保留英文。

## 上游与下游

- 内容入口：[知识库](../知识库/README.md)
- 版本入口：[版本记录](../版本记录/README.md)
- 项目入口：[项目实战](../项目实战/README.md)
- 仓库说明：[仓库说明](../README.md)
