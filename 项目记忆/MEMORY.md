# 项目记忆索引

本文件只提供长期记忆入口，不保存长计划、聊天记录或重复正文。正式规则以 `AGENTS.md`、`CODEX-PRD.md` 和项目管理记录为准。

## 索引

- [decisions.md](decisions.md)：活跃决策摘要及正式 ADR 链接。
- [patterns.md](patterns.md)：可复用的协作与交付模式。
- [resources.md](resources.md)：安全资源位置和官方入口。
- [open-questions.md](open-questions.md)：非阻塞决策门与已确认实施决定。
- [deprecated.md](deprecated.md)：被替代规则和迁移说明。

## 安全边界

任何项目记忆文件都不得保存密码、Token、Cookie、私钥、云 AccessKey、个人敏感信息或其他凭据。
