# 10 数据服务

- **阶段编号：** 10
- **阶段名称：** 数据服务

## 阶段目标

建立维护企业项目常用数据组件的基础能力。

## 主要主题

- SQL 基础
- MySQL
- Redis
- RabbitMQ/Kafka 基础
- 备份恢复
- 复制
- 高可用
- 性能基础
