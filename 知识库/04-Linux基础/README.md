# 04 Linux 基础

- **阶段编号：** 04
- **阶段名称：** Linux 基础

## 阶段目标

建立独立完成 Linux 主机基础操作与服务管理的能力。

## 主要主题

- 目录
- 文件
- 文本
- 用户
- 权限
- 进程
- 软件包
- 服务
- 计划任务

## 前置阶段

- [03 实验环境](../03-实验环境/README.md)

## 阶段产出

能独立操作一台 Linux 主机。
