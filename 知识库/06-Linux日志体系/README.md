# 06 Linux 日志体系

- **阶段编号：** 06
- **阶段名称：** Linux 日志体系

## 阶段目标

建立在单机环境中定位、管理和维护日志的能力。

## 主要主题

- journald
- rsyslog
- 服务日志
- audit
- logrotate
- 日志故障

- [05 Linux 系统管理](../05-Linux系统管理/README.md)
