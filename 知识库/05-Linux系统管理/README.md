# 05 Linux 系统管理

- **阶段编号：** 05
- **阶段名称：** Linux 系统管理

## 阶段目标

建立完成服务器初始化、维护与恢复的系统管理能力。

## 主要主题

- 启动
- 磁盘
- LVM
- 文件系统
- 网络
- 时间
- 内核参数
- 备份恢复

## 阶段产出

能完成服务器初始化和日常维护。
