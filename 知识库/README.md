# 知识库

## 目录职责

本目录按 00 至 20 的成长阶段组织正式知识章节，并为导航、实验、项目和辅助内容提供稳定引用目标。

## 权威内容类型

`知识库/` 是正式知识正文的唯一权威来源。概念、原理、平台、服务、工具、语言、云服务、硬件和方法论正文只在对应阶段维护一份。

## 非范围

- 不复制学习路线或技术索引；
- 不保存实验步骤、项目实现、独立故障案例或面试题；
- 本任务不创建任何正式知识章节。

## 命名规则

- 阶段目录使用 `NN-阶段名称`，编号范围为 00 至 20；
- 阶段入口固定为 `README.md`；
- 技术名保留英文，名称不含空格并以短横线连接；
- 后续章节文件的细化命名不在本任务定义。
