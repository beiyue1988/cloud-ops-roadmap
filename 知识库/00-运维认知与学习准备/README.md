# 00 运维认知与学习准备

- **阶段编号：** 00
- **阶段名称：** 运维认知与学习准备

## 阶段目标

建立对运维岗位、企业系统位置、学习方式与实验安全边界的基础认知。

## 主要主题

- 岗位
- 网站访问链路
- 企业 IT 架构
- 学习方法
- 实验安全

## 前置阶段

无。

## 阶段产出

能说明运维在企业系统中的位置。
