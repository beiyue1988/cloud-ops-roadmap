# 02 网络基础

- **阶段编号：** 02
- **阶段名称：** 网络基础

## 阶段目标

建立理解基础网络通信并定位常见连通性问题的能力。

## 主要主题

- OSI/TCP-IP
- IPv4
- 子网
- 路由
- NAT
- DNS
- DHCP
- TCP/UDP
- HTTP/HTTPS
- 抓包

## 前置阶段

- [00 运维认知与学习准备](../00-运维认知与学习准备/README.md)

## 阶段产出

能完成基础连通性排查。
