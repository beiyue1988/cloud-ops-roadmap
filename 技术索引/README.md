# 技术索引

## 目录职责

本目录按技术名称、问题类型和岗位能力组织查询入口，帮助读者定位权威内容。

## 权威内容类型

- 技术主题与权威内容位置之间的索引关系；
- 面向查询场景的分类入口与交叉导航。

本目录仅导航，不复制正文；正式知识正文以[知识库](../知识库/README.md)为唯一权威来源。

## 非范围

- 不保存正式知识正文；
- 不保存命令教程、实验步骤、故障处置正文或面试答案；
- 不在本任务中创建具体索引文件。

## 命名规则

- 目录入口固定为 `README.md`；
- 后续索引文件使用中文分类名称，技术名保留英文，文件名不含空格并以短横线连接。
