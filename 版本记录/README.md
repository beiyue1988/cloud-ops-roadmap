# 版本记录

## 目录职责

本目录管理项目采用的软件版本基线、兼容范围与内容更新记录的入口。

## 权威内容类型

软件版本基线、版本敏感点与项目更新日志以本目录为权威归属。

## 当前入口

- [软件版本基线](软件版本基线.md)：集中维护治理状态、证据状态、锁定粒度、阶段门与当前版本值；
- [更新日志](更新日志.md)：追加记录基线建立、升级、回退和验证状态变化。

## 非范围

- 不复制技术正文、安装步骤或外部发布说明；
- 不在目录入口复制完整版本主表；
- 不把来源核验写成环境或集成验证。

## 命名规则

- 目录入口固定为 `README.md`；
- 后续文件名使用明确的版本或记录职责名称，不含空格并以短横线连接。

## 上游与下游

- 仓库说明：[仓库说明](../README.md)
- 内容入口：[知识库](../知识库/README.md)
- 来源入口：[参考资料](../参考资料/README.md)
- 规范入口：[企业规范](../企业规范/README.md)

## 当前阶段

Phase 0 已建立首份版本与来源基线；后续变更必须同步记录证据、影响和回退边界。
