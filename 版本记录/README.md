# 版本记录

## 目录职责

本目录管理项目采用的软件版本基线、兼容范围与内容更新记录的入口。

## 权威内容类型

未来软件版本基线、版本敏感点与项目更新日志以本目录为权威归属。

## 非范围

- 不复制技术正文、安装步骤或外部发布说明；
- 不在本任务中核验、改变或新增软件版本基线；
- 本任务不创建版本基线文件或更新日志。

## 命名规则

- 目录入口固定为 `README.md`；
- 后续文件名使用明确的版本或记录职责名称，不含空格并以短横线连接。
