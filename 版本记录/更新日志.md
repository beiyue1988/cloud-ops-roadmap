# 版本基线更新日志

## 用途与规则

本文件追踪 `软件版本基线.md` 的建立、升级、降级、回退和证据状态变化。记录只追加，不覆盖或删除历史；错误记录以新条目纠正并链接原条目。外部 Release Notes 只作为证据链接，不复制到本文件。

major、minor、固定 tag 或架构组合变化必须由新 ADR 或替代 ADR 批准。仅 patch 变化且兼容边界未改变时，可按基线流程更新，但必须保留来源核验与环境验证状态。

## 变更记录字段

每条记录必须包含：日期、组件、旧值、新值、变更类型、原因、证据、验证状态、影响范围、关联任务/ADR、回退说明。

### 2026-07-15：锁定 VMware Workstation Pro 26H1 实验基线

| 字段 | 内容 |
|---|---|
| 日期 | 2026-07-15 |
| 组件 | VMware Workstation Pro 阶段 03 本地虚拟化线 |
| 旧值 | `phase-gated` / `not-yet-verified`；产品、版本和宿主机兼容性进入对应任务前核定 |
| 新值 | `locked` / `source-verified`；VMware Workstation Pro 26H1，Windows 10 20H1+ 或 Windows 11 的 x86-64 宿主为主教学路径，Linux 宿主按官方发行版/内核矩阵逐次核验；下载 build 动态维护 |
| 变更类型 | minor 等级的日历发布线与架构组合决定；治理状态和证据状态变化 |
| 原因 | 26H1 已于 2026-05-14 GA，Windows Workstation Pro 转为 64 位，官方宿主矩阵、下载入口、虚拟硬件版本、客体工具和许可边界能够形成来源证据闭环 |
| 证据 | [ADR-0006](../项目管理/决策记录/ADR-0006-VMware-Workstation-Pro-26H1实验基线.md)与[软件版本基线](软件版本基线.md)中 2026-07-15 的 VMware/Rocky 直接来源记录 |
| 验证状态 | 产品、发布线、宿主支持、vHW 22、`open-vm-tools` 与许可边界为 `source-verified`；TechDocs Release Notes 自动访问返回 403 并保留限制；未下载、安装或运行，尚非 `environment-verified` |
| 影响范围 | 阶段 03 章节清单与后续本地实验环境；不修改 Rocky Linux、Kubernetes、GPU、VitePress、RuoYi 或其他既有基线 |
| 关联任务/ADR | P1-02-03G；[ADR-0006](../项目管理/决策记录/ADR-0006-VMware-Workstation-Pro-26H1实验基线.md) |
| 回退说明 | 若 26H1 下载不可用、官方支持交集消失或首个实验无法在不降低安全基线的前提下复现，保留本条并追加回退记录，从实验前快照恢复；改变日历发布线或主产品必须使用替代 ADR，不静默切换到 25H2 或其他工具 |
