# 脚本资源

## 目录职责

本目录归档未来可复用、可验证并具有明确用途的运维脚本与自动化资源。

## 权威内容类型

未来 Shell、Python、Ansible、Docker、Kubernetes 与 Terraform/OpenTofu 相关脚本或代码资源以本目录为权威归属。

## 非范围

- 不保存知识正文、实验步骤、项目说明或真实凭据；
- 不保存无法说明用途、输入、输出和验证方式的代码；
- 本任务不创建语言子目录、脚本、配置或代码文件。

## 命名规则

- 目录入口固定为 `README.md`；
- 后续资源名称不含空格并以短横线连接，语言和技术名保留英文；
- 语言子目录和文件级规则不在本任务定义。
